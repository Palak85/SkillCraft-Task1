# Responsive Interactive Landing Page

A modern and fully responsive landing page website built using **HTML, CSS, and JavaScript**.
This project includes an interactive fixed navigation bar with hover effects, scroll-based styling changes, smooth animations, and mobile responsiveness.

---

## 🚀 Project Overview

This project was developed as part of an internship task to demonstrate frontend development skills including:

* Responsive Web Design
* Interactive Navigation Menus
* JavaScript DOM Manipulation
* Smooth UI Animations
* Mobile-Friendly Layouts

The landing page is designed with a clean modern UI and works across desktop, tablet, and mobile devices.
